## Installation



1. Install dependences

```bash
npm run i
```

2. Local deploy

first run the start local deploy for modify the web app
```bash
npm run start

```

then run the backend which is deployed on port 3000

```bash
npm run nodemon

```


## Usage

run the command

```bash
npm run build

```
to create the project build in [pip](dist) folder
